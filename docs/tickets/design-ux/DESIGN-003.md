# DESIGN-003 — Remove global transition strategy and scope motion

- **Status:** todo
- **Priority:** P1
- **Topic:** Design & UX

## Problem

Global transitions and widespread decorative motion create cost and visual noise.

## Goal

Adopt a deliberate motion system.

## Proposed Work

- Remove global transition rules.
- Audit animated surfaces.
- Keep motion only where it helps hierarchy/feedback.

## Done When

- Motion feels intentional and performance risk is lower.
