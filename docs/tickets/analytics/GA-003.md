# GA-003 — Clarify pageview semantics for language/content state

- **Status:** todo
- **Priority:** P2
- **Topic:** Analytics

## Problem

Path-only pageview logic may not fully represent language/content state.

## Goal

Define how the app should report pageviews when language changes without a URL change.

## Proposed Work

- Decide event strategy.
- Align title/page path/language dimensions.
- Update reporting assumptions.

## Done When

- Analytics consumers can interpret page traffic correctly by language/content state.
