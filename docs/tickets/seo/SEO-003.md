# SEO-003 — Add explicit 404 / noindex behavior

- **Status:** todo
- **Priority:** P1
- **Topic:** SEO

## Problem

The SPA can expose soft-404 indexation risk.

## Goal

Ensure unknown routes are handled in a search-safe way.

## Proposed Work

- Review 404 rendering path.
- Add explicit noindex semantics for not-found states.
- Review canonical behavior for fallback routes.

## Done When

- Unknown URLs no longer behave like normal indexable pages.
