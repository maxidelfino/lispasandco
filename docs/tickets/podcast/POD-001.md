# POD-001 — Define homepage teaser vs dedicated podcast page

- **Status:** done
- **Priority:** P0
- **Topic:** Podcast

## Problem

The homepage and `/podcast` currently overlap too much.

## Goal

Assign one clear role to each surface.

## Proposed Work

- Define what the homepage teaser includes.
- Define what only `/podcast` should own.
- Restore/clarify navigation path.

## Done When

- The podcast feature has clear IA and no longer duplicates itself.

## Resolution

- The dedicated `/podcast` page is now the clear podcast destination.
- Current navigation points users directly to `/podcast`.
- The homepage no longer carries a competing podcast surface that blurs ownership.
