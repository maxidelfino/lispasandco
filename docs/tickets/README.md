# Tickets

This directory stores local markdown tickets for the solo maintenance/refactor workflow.

## Status Model

- `todo`
- `in-progress`
- `blocked`
- `done`

## Priority Model

- `P0` — critical foundation
- `P1` — high-value / high-urgency
- `P2` — important but not blocking
- `P3` — polish / follow-up
